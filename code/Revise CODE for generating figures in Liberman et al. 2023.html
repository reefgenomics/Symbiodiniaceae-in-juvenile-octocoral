Author

This code is used to analyze Symportal results

In this code I use the raw file outputs for profiles from Symportal and also the raw profile of adult colonies that were already analyzed in Liberman et al. (2022) to compare the algal symbiont composition in Rhytisma fulvum juveniles across depth during 11 month experiment.
